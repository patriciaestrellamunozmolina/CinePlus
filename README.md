# CinePlus
Aplicacion basica de cobro de cine creada en python con wxglade. consta con 4 ventanas incio de sesion sin conexion a base de datos, Taquilla, Dulceria y ticket
Creada por alumnos de educacion media superior (Bachillerato)

Ventana de incio de sesión (Login)
- El sistemadebe tener una pantalla de inicio de sesión.
- Para este proyecto, el inicio de sesión se hará con datos preesablecidos(por eejemplo,usuario:cajero y contraseña:12345).No es necesario implementar una base de datos.

Venta de boletos(Taquilla)
-el sistema debe permitir seleccionar la pelicula, el horario y la sala 
"Megan 2.0" 14:00,17:00,20:00 Sala 1
"10 cosas que odio de ti" 13:30,16:30,19:30 Sala 2
"La hermanastra" 15:00,18:00,21:00 Sala 3

-Se debe de especificar la cantidad de boletos.
-Se debe indicar si los boletos son para adultos o menores de edad, ya que el precio es diferente (Adultos $90, menores de edad $80)
-al finalizar la selección de boletos , el sistema debe mostrar un resumen de los boletos adquiridos y calcular el subtotal de taquilla.

Ventana de productos(Dulceria)
-Despues de la taquilla, el sistema debe preguntar si el cliente desesa comprar productos de dulceria.
-si la respuesta es afirmativa, el sistema debe llevar al empleado a una nueva pantalla.
-En esta pantalla se debe mostrar tres combos preestablecidos:
"Basico" $75 Palomitas chicas + refresco chico (imagen alusiva al combo)
"Cuates" $130 Palomitas medianas + 2 refrescos medianos 
"Familiar" $200 Palomitas grandes + 3 refrescos grandes + dulces 
-por cada combo, el sistema debe mostrar su precio, el contenido y una imagen alusiva.
-Tambien se debe permitir la venta de producos individuales:
-Palomitas:Tamaño chico $30, mediano $50 y grande $70
Refrescos:Sabores Cola, Sprite,Manzana, en tamaño chico $25 , mediano $40 , y grande $60.
- El empleado debe poder indicar la cantidad de cada combo op producto individual .

  Resumen de la compra y cobro(Ticket)
  -Al finalizar la selección de productos de dulceria, el sistema debe mostrar una ventana de resumen.
  Esta ventana debe incluir un desglose de la compra:
  -Sección "Taquilla" con los boletos comprados y su subtotal.
  -Sección "Dulceria" con los combos y productos individuales , y su subtotal.
  -Finalmente el sistema debe mostrar el costo total de toda la compra (Taquilla+ dulceria).
